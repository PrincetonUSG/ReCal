We're Sorry :(

We're working hard to ensure this never happens again.

Around midnight, ReCal experienced a database error, resulting in yesterday's changes to schedules being deleted. We have since restored our database to the latest backup at 04/07/2015 at 00:27, or Tuesday morning. The database is now back online, and any changes you made before Tuesday morning, 00:27 AM, have been preserved. We are investigating the cause of the data loss, and we currently suspect corruption.

Meanwhile, we are launching an upgraded backup system that will prevent this from happening again. The key improvement is that we now take hourly backups. We know that we have failed your trust, but please know that we will be working tirelessly to regain it, and we are doing everything to ensure that this never, ever, happens again.

Humblest regrets,
